1.1. versions

    • elasticseach plugin Kopf
  • logstash 2.4.6
  • kibana 4.x
  • oracle JDK java 1.8.x

1.2. 安装和配置

1.2.1. elasticsearch

下载

《Elasticsearch: 权威指南》

安装并运行 Elasticsearch

$ tar zxf elasticsearch-2.4.6.tar.gz

关键配置

export ES_HEAP_SIZE=2G

cluster.name=product
